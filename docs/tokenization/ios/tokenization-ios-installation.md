# Installation

Use CocoaPods  to install the Fattmerchant iOS SDK.

1. Install [CocoaPods](https://guides.cocoapods.org/using/getting-started.html)
2. Add `pod 'Fattmerchant'` to your `Podfile`
3. Run `pod install`
