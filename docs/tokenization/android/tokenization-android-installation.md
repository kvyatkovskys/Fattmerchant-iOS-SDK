# Installation

## Jitpack
To install,

1. Add the JitPack repository to your build file

    ```
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
    ```

2. Add the following line to your `build.gradle` file

    `implementation 'com.github.fattmerchantorg:fattmerchant-android-sdk:v1.0.4'`
